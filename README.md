# Currency Converter

A React + TypeScript currency converter with:

- Instant conversion with **swap functionality**
- **Popular currencies board**
- **Conversion history** (last 5 operations)
- **Interactive charts** for weekly currency evolution using Recharts
- **Offline mode** with cached rates in `localStorage`
- **GSAP animations** for transitions

Built with **React (Vite)**, **Redux Toolkit**, **Tailwind CSS**, **GSAP**, and **React Icons**.

---

## Installation

1. Clone the repository:
```bash
  git clone https://vyllian/github.io/currency-converter/
```

2. Install dependencies:
```bash
    npm install
```

3. Run project localy:
```bash
    npm run dev
```

4. FDeploy to Github pages:
```bash
    npm run deploy
```
